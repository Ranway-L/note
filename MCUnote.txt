自定义函数： 延时函数delay（毫秒级）
void delay(unsigned int z)
{
    unsigned int x, y;
    for(x = z; x > 0; x-- )
        for(y = 114; y > 0 ; y--);
}

标准库函数： intrins.h
内部函数
字符型循环左移： _crol_
字符型循环右移： _cror_
	#include <intrins.h>
	void test_crol (void)
	{
		unsigned char a
		unsigned char b
		a = 0xFE; // 11111110
		b = _crol_(a,1); // b now is 0xFD, 二进制形式11111101
	}

蜂鸣器
	有源蜂鸣器 （价格较贵）
		内部含震荡源，通电就可以工作

	无源蜂鸣器
		内部不含震荡源，用2K~5K方波驱动才可以工作

三极管
	E： EMITTER(发射极)
	B： BASE（基极）
	C： COLLECTOR(集电极)

	PNP管：
		E级到B级电流大于1ma或电压大于0.7V则三极管导通

	NPN管:
		B级到E级电流大于1ma或电压大于0.7V则三极管导通
		

LED灯：尖头为阴极，平头为阳极
		
数码管
	
	共阴极
		黑表笔接GND，红表笔接管，管亮
	共阳极
		红表笔接GND，黑表笔接管，管亮
	#红表笔输出正电压
	#中间的脚为GND公共脚
	#几位数码管就是几个8
	#L为低位，H为高位

锁存器：把数据输入端与输出端进行隔离或链接 
	OE：
		相当于锁存器开关
		接H时高阻态，锁存器不导通
		接L（接地）时锁存器导通
	D：输入端
	Q：输出端
	LE：
		H：Q随D动态变化
		L：Q保持当前状态
